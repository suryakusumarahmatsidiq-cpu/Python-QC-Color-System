# 🎨 Automated Color-Matching System for Paint Manufacturing

**Oleh:** Rahmat Sidiq Suryakusuma, S.Tr. Kimia Analis  
**Role:** Quality Control (QC) & Data Enthusiast  
**Tech Stack:** Python, Pandas, Data Analysis, Microsoft Excel  

---

## 📌 Latar Belakang Masalah (*The Problem*)
Di lingkungan produksi cat, penyesuaian warna (*color-matching*) dan *troubleshooting* hasil produksi sering kali mengandalkan metode manual (*trial and error*). Pendekatan ini memakan waktu tunggu (*downtime*) yang cukup lama di laboratorium *Quality Control* (QC) dan berisiko menyebabkan pemborosan bahan baku (*material waste*) jika formula koreksi tidak akurat. 

## 💡 Solusi yang Dikembangkan (*The Solution*)
Sebagai inisiatif untuk meningkatkan efisiensi di departemen QC, saya mengembangkan skrip otomatisasi menggunakan **Python**. Sistem ini dirancang untuk:
1. **Mengolah Data Historis:** Menarik dan menganalisis kumpulan data riwayat produksi cat sebelumnya.
2. **Analisis Parameter Spesifik:** Memperhitungkan variabel kunci produksi, khususnya *color strength* (kekuatan warna) dan parameter QC lainnya.
3. **Rekomendasi Cerdas:** Mengalkulasi dan merekomendasikan formula warna (penyesuaian resin, pigmen, dll.) yang paling optimal secara instan untuk memperbaiki *batch* produksi.

## 🚀 Dampak Bisnis & Operasional (*The Impact*)
* **Standarisasi Proses (Process Control):** Mengubah proses *matching* warna dari tebakan manual menjadi keputusan berbasis data analitik yang terstandardisasi dan konsisten.
* **Efisiensi Waktu Produksi:** Memangkas waktu *troubleshooting* warna secara drastis, sehingga operasional mesin produksi berjalan lebih optimal.
* **Pengendalian Material (Material Planning):** Meminimalisir terbuangnya bahan baku akibat proses koreksi yang berulang, mendukung efisiensi *inventory* dan rantai pasok pabrik.
